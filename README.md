# Crime Reporting and Analysis System

## Description
A web-based system for reporting crimes, managing case status, and improving citizen safety.

## Features
- User Authentication (Citizen, Officer, Admin)
- Crime Report Submission with Evidence
- Case Status Tracking
- Secure Access Control

## Tech Stack
- PHP
- MySQL
- HTML, CSS (Tailwind)
- XAMPP

## Setup
1. Clone the repo
2. Import database
3. Configure db.php
4. Run using XAMPP
